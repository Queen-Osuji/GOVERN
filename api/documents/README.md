## Security Notes

- Keep file sizes reasonable (under 25MB per file for email attachments)
- Use PDF format for best compatibility
- Ensure files are properly formatted and readable
- Consider password protection for premium content

## Usage

When a customer purchases a product, the system will:
1. Verify the payment
2. Find the corresponding PDF file
3. Send it as an email attachment directly to the customer
4. Log the transaction

This eliminates the need for download links and provides immediate delivery.
